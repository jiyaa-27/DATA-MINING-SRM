# Foundations of Data – Data Mining 

This repository contains a PowerPoint presentation covering **Unit 1 of Data Mining**.  
The PPT explains the **basic concepts of data**, its **types**, **patterns**, **visualization**, and **data mining techniques** in a simple and easy-to-understand manner.

---

## 📌 Topics Covered

- Introduction to Data  
- Importance of Data in Data Science & Machine Learning  
- Types of Data  
  - Structured Data  
  - Semi-Structured Data  
  - Unstructured Data  
- Data Patterns  
  - Association  
  - Sequential  
  - Classification  
  - Clustering  
  - Anomaly Detection  
- Data Objects and Attributes  
- Attribute Types  
  - Categorical  
  - Numerical  
- Measurement Scales  
  - Nominal  
  - Ordinal  
  - Interval  
  - Ratio  
- Data Visualization  
  - Bar Chart  
  - Line Chart  
  - Pie Chart  
  - Histogram  
  - Scatter Plot  
- Introduction to Data Mining  
- Data Mining Techniques  
  - Classification  
  - Clustering  
  - Regression  
  - Association Rules
